# AI Institution Search - Frontend

A modern React application for finding educational institutions and courses with AI-powered recommendations.

## Features

- **Unified Role-Based Authentication**: Single login page that handles admin, provider, and user roles
- **Role-Specific Dashboards**:
  - **Admin Dashboard**: Manage course data and institutions
  - **Provider Dashboard**: Add and manage institution information
  - **User Home**: Search courses and get AI recommendations
- **Modern UI**: Built with Tailwind CSS and Framer Motion animations
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Authentication Flow

The application uses a unified login system (`/login`) that:
1. Accepts username and password
2. Calls the backend API to authenticate
3. Receives user role information
4. Redirects to appropriate dashboard based on role:
   - `admin` → `/admin/dashboard`
   - `provider` → `/provider/dashboard` 
   - `user` → `/user/home`
